# Hort — Logo Prompts

## Concept

Hort = der Nibelungenhort, der Schatz. Ein Ort wo Wertvolles sicher verwahrt wird.

Visual: Minimalistisch, tech-nah, deutsch-mythologisch. Kein Cartoon, kein Clipart. Passt zu den anderen s16e Tools (Spürhund, Streifen, Fachwerk).

## Gemini (Imagen 3)

### Prompt 1 — Minimal Icon

```
Minimalist app icon for a CLI tool called "Hort" (German for treasure hoard). A stylized vault door or treasure chest rendered as a clean geometric symbol. Monochrome with a single accent color (gold or amber). Flat design, no gradients. Dark background. Tech/developer aesthetic. SVG-friendly simplicity.
```

### Prompt 2 — Mythological

```
Minimalist logo mark combining a dragon's eye with a keyhole. Inspired by the Nibelungenhort (German treasure mythology). Clean geometric lines, suitable for a developer CLI tool. Gold accent on dark background. No text. Icon only. Flat design, high contrast.
```

### Prompt 3 — Runic

```
A runic-style monogram of the letter H, inspired by Elder Futhark runes and German mythology. Geometric, angular, minimal. Suitable as a developer tool icon. Gold or amber on black. No decorative elements. Clean edges, works at 16x16 and 512x512.
```
